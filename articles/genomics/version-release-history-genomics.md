# <a name="version-release-history"></a>Versiegeschiedenis van release
De client voor Microsoft Genomics Python voor oplossingen en nieuwe functies wordt regelmatig bijgewerkt door het team van Microsoft Genomics. 

## <a name="latest-release"></a>Meest recente versie
De huidige client voor Python is versie 0.7.2. Publicatie 1 November 2017. Hierbij wordt gebruikgemaakt van GATK 3.3. 


## <a name="release-history"></a>Versiegeschiedenis 
Nieuwe versies van de client voor Microsoft Genomics Python worden uitgebracht over twee keer per jaar. Als nieuwe versies van de client voor Microsoft Genomics Python worden vrijgegeven, wordt hier een lijst met oplossingen en functies bijgewerkt. Wanneer er nieuwe versies worden uitgebracht, blijven eerdere versies moeten voor ten minste 90 dagen wordt ondersteund. Als eerdere versies zijn niet meer wordt ondersteund, wordt deze aangeduid op deze pagina.
