# <a name="azure-toolkit-for-eclipse"></a>Azure Toolkit voor Eclipse
De Azure-werkset voor Eclipse biedt sjablonen en functionaliteit die u kunt gemakkelijk maken, ontwikkelen, testen en implementeren van Azure-toepassingen met behulp van de Eclipse integrated development environment (IDE). Het is een open source-project waarvan de broncode is beschikbaar onder de [MIT-licentie van het project site](https://github.com/microsoft/azure-tools-for-java).

> [!NOTE]
> Er is ook een Azure-werkset voor de IntelliJ IDE. Zie voor meer informatie [Azure Toolkit voor IntelliJ](azure-toolkit-for-intellij.md).
> 
>
