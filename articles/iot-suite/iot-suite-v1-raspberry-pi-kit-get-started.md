# <a name="connect-your-microsoft-azure-iot-raspberry-pi-3-starter-kit-to-the-remote-monitoring-solution"></a>Verbinding maken met uw Microsoft Azure IoT frambozen Pi 3 Starter Kit de oplossing voor externe controle

De zelfstudies in deze sectie kunnen u meer informatie over hoe een frambozen Pi 3-apparaat verbindt met de oplossing voor externe controle. De zelfstudie geschikt is voor de gewenste programmeertaal kiezen en de of u hebt de sensor hardware beschikbaar voor gebruik met uw frambozen Pi.

## <a name="the-tutorials"></a>De zelfstudies

| Zelfstudie | Opmerkingen | Talen |
| -------- | ----- | --------- |
| Gesimuleerde telemetrie (basis)| Sensorgegevens simuleert. Maakt gebruik van een zelfstandige frambozen Pi. | [C][lnk-c-simulator], [Node.js][lnk-node-simulator] |
| Echte temperatuursensor (tussenliggend) | Maakt gebruik van gegevens uit een sensor BME280 is verbonden met uw frambozen Pi. | [C][lnk-c-basic], [Node.js][lnk-node-basic] |
| Implementeer firmware-update (Geavanceerd)| Maakt gebruik van gegevens uit een sensor BME280 is verbonden met uw frambozen Pi. Hiermee kunt externe firmware-updates op uw frambozen Pi. | [C][lnk-c-advanced], [Node.js][lnk-node-advanced] |
