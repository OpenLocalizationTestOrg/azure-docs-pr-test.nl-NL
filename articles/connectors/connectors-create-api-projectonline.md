# <a name="get-started-with-hello-projectonline-connector"></a>Aan de slag met Hallo ProjectOnline connector
Project Online is een flexibele online oplossing voor beheer van de portfolio project (PPM) en de dagelijkse werk van Microsoft. Bezorgd via Office 365, Project Online kunnen organisaties tooget snel de slag met krachtige project management mogelijkheden tooplan prioriteren en projecten en project portfolio investeringen te beheren, vanaf vrijwel elke locatie op vrijwel elk apparaat.

U kunt aan de slag door het maken van een logische app nu, Zie [een logische app maken](../logic-apps/logic-apps-create-a-logic-app.md).
