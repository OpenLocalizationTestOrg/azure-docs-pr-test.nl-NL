# <a name="related-and-linked-resources-in-hello-tile-gallery"></a>Verwante en gekoppelde resources in Hallo tegel galerie
Hallo tegel galerij kunt u toofind tegels voor een bepaalde bron en sleep deze naar uw huidige blade. Hallo tegel galerie gebruikt, kunt u beheerweergaven die resources omvatten. Voor een opgegeven resource gerelateerde Hallo resources omvatten alle Hallo resources in de resourcegroep en alle resources die een koppeling tooor van Hallo-resource.

## <a name="linked-resources-in-resource-manager"></a>Gekoppelde resources in het Resource Manager
Koppelen is een functie van Hallo Resource Manager.  Hiermee kunt u toodeclare relaties tussen resources zelfs als ze zich niet in Hallo dezelfde resourcegroep. Koppelen, heeft geen invloed op Hallo runtime van uw resources, niet van invloed op de facturering en niet van invloed op rollen gebaseerde toegang.  Het is gewoon een mechanisme kunt u toorepresent relaties zodat hulpprogramma's als Hallo tegel galerie u een geavanceerde beheermogelijkheden krijgt optreden.  De hulpprogramma's kunnen inspecteren Hallo koppelingen met Hallo koppelingen API en geef aangepaste relatie management ook optreedt. 

## <a name="how-do-i-link-my-resources"></a>Hoe kan ik mijn resources koppelen?
Wanneer u resources via Hallo portal of met het implementeren van een sjabloon via Azure PowerShell of Azure CLI maakt, worden automatisch koppelingen gemaakt voor enkele afhankelijke resources. U kunt ook programmatisch resources koppelen met behulp van Hallo [gekoppelde Resources REST-API](/rest/api/resources/resourcelinks).
