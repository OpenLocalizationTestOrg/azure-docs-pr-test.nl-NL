# <a name="web-app-on-linux-documentation"></a>Web-App op Linux-documentatie
Web-App op Linux is momenteel in de openbare Preview en maakt toorun klanten hun web-apps systeemeigen op een Linux-platform. Hiermee kunt u betere compatibiliteit met bepaalde soorten toepassingen en maakt het eenvoudiger toomigrate bestaande web-apps die worden gehost op een Linux-platform elders naar Azure App Services.

Klanten kunnen maken web-apps op Linux alleen voorkomen binnen een Linux ingeschakeld App Service-Plan en Linux-gebaseerde web-apps kunnen alleen hosten dit App Service-Plan. 

> Openbare Preview prijscategorie voor Web-App op Linux is 50% van de normale prijs Hallo en Hallo prijs wordt berekend door het halveren Hallo aantal rekenuren gebruikt door uw App Service-Plan.
> 
> Er is geen SLA wordt aangeboden tijdens Hallo openbare preview-periode.
