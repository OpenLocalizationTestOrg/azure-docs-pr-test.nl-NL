# <a name="web-app-on-linux-documentation"></a>Web-App op Linux-documentatie
Web-App op Linux is momenteel in de openbare Preview en kan klanten hun web-apps systeemeigen uitvoeren op een Linux-platform. Hiermee kan voor betere compatibiliteit met bepaalde soorten toepassingen en gemakkelijker te migreren van bestaande web-apps die worden gehost op een Linux-platform elders naar Azure App Services.

Klanten kunnen maken web-apps op Linux alleen voorkomen binnen een Linux ingeschakeld App Service-Plan en Linux-gebaseerde web-apps kunnen alleen hosten dit App Service-Plan. 

> Openbare Preview prijscategorie voor Web-App op Linux is 50% van de normale prijs en de prijs wordt berekend door het aantal rekenuren gebruikt door uw App Service-Plan halveren.
> 
> Er is geen SLA wordt aangeboden in de openbare preview-periode.
