# <a name="view-all-hello-enterprise-apps-that-i-can-manage-in-azure-active-directory"></a>Alle Hallo zakelijke apps die ik kunt beheren in Azure Active Directory weergeven
U kunt uw bedrijfstoepassingen op Hallo Azure Active Directory (Azure AD) kunt beheren. Dit omvat weergeven Hallo-apps die u kunt beheren, gebruikers of groepen tooan app toe te wijzen, onderhouden van de eigenschappen voor Hallo-app, zoals Hallo toepassing naam/het logo en zelfs een toepassing uitschakelen zodat er geen gebruikers tooit kunnen aanmelden.

## <a name="how-do-i-view-all-my-apps"></a>Hoe kan ik alle mijn apps bekijken?
1. Meld u aan toohello [Azure-portal](https://portal.azure.com) met een account met globale beheerdersrechten voor Hallo-directory.
2. Selecteer **meer services**, voer **Azure Active Directory** in het tekstvak Hallo en selecteer vervolgens **Enter**.
3. Op Hallo **Azure Active Directory -** ***directoryname*** blade (dat wil zeggen hello Azure AD blade voor Hallo-directory die u beheert), selecteer **bedrijfstoepassingen**.

    ![Openen zakelijke apps](./media/active-directory-coreapps-view-azure-portal/open-enterprise-apps.png)
4. Op Hallo **bedrijfstoepassingen** blade Selecteer **alle toepassingen**. Hallo die u (bijvoorbeeld alleen uitgeschakelde tooview apps wilt) app-lijst toofind vanaf deze blade apps toomanage selecteert, Hallo weergegeven kolommen wijzigen of filteren.
