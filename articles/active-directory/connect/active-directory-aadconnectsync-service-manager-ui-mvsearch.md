# <a name="sync-service-manager-metaverse-search"></a>Sync-Service Manager Metaverse zoeken

![Sync-Service Manager](./media/active-directory-aadconnectsync-service-manager-ui/mvsearch.png)

Hallo metaverse zoeken tabblad is nuttig voor het oplossen van problemen met gegevens. In het bovenste gedeelte hello, kunt u een query op basis van een combinatie van kenmerken. Wanneer u tevreden met uw query bent, klikt u op **Search**. Hallo-resultaat is zichtbaar in Hallo onder raster. U kunt selecteren welke kolommen moeten worden weergegeven met **kolominstellingen**.

Selecteer een object in de zoekresultaten Hallo en **eigenschappen** toosee hello [eigenschappen metaverse-object](active-directory-aadconnectsync-troubleshoot-object-not-syncing.md#metaverse-object-properties).
