# <a name="add-or-change-work-information-for-a-user-in-azure-active-directory"></a>Toevoegen of wijzigen van gegevens van het werk voor een gebruiker in Azure Active Directory
Dit artikel wordt uitgelegd hoe tooadd of wijzig werkgegevens zoals telefoonnummers of afdeling namen voor een gebruiker in Azure Active Directory (Azure AD). Zie voor meer informatie over het toevoegen van nieuwe gebruikers in uw organisatie [toevoegen van nieuwe gebruikers tooAzure Active Directory](active-directory-users-create-external-azure-portal.md).

## <a name="toochange-work-information"></a>Werkinformatie toochange
1. Meld u aan toohello [Azure-portal](https://portal.azure.com) met een account met globale beheerdersrechten voor Hallo-directory.
2. Selecteer **meer services**, voer **gebruikers en groepen** in het tekstvak Hallo en selecteer vervolgens **Enter**.

   ![Gebruikersbeheer openen](./media/active-directory-users-work-info-azure-portal/create-users-user-management.png)
3. Op Hallo **gebruikers en groepen** blade Selecteer **gebruikers**.

   ![Blade openen Hallo-gebruikers](./media/active-directory-users-work-info-azure-portal/create-users-open-users-blade.png)
4. Op Hallo **gebruikers en groepen - gebruikers** blade, selecteert u een gebruiker in de lijst Hallo.
5. Selecteer op de blade voor de geselecteerde gebruiker Hallo Hallo **contactgegevens werk**.

    ![Werkinformatie openen](./media/active-directory-users-work-info-azure-portal/active-directory-create-users-work-info.png)
6. Toevoegen of wijzigen van Hallo werkgegevens. Selecteer in de opdrachtbalk Hallo **opslaan**.
