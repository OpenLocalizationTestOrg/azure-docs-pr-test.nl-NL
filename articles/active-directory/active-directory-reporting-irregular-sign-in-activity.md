# <a name="irregular-sign-in-activity"></a>Onregelmatige aanmeldingsactiviteiten
Onregelmatige aanmeldingen zijn bestanden die zijn geïdentificeerd door onze machine learning-algoritmen, op basis van een 'onmogelijke reis' voorwaarde gecombineerd met een afwijkende teken op de locatie en het apparaat Hallo. Dit kan erop wijzen dat een hacker heeft aangemeld met dit account.
