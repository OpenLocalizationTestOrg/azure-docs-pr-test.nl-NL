# <a name="azure-active-directory-b2b-collaboration-invitation-redemption"></a>Azure Active Directory B2B-samenwerking uitnodiging inwisseling

## <a name="azure-ad-and-microsoft-account-users"></a>Azure AD en gebruikers van Microsoft-account
De ervaring inwisseling is voor gebruikers met een bestaande Azure AD-account of Microsoft-accounts, net zo eenvoudig als het aanmelden.

## <a name="social-id-user-first-time-redemption"></a>Sociale ID gebruiker eerst inwisseling
Azure AD B2B-samenwerking maakt het eenvoudig voor elk e-mailadres voor inwisseling moet worden gebruikt. Bekijk de ervaring inwisseling wanneer een niet-Microsoft-e-mailadres wordt gebruikt voor B2B-samenwerking. Deze stroom inwisseling is gecompliceerder, omdat u mogelijk moet u een account maken op het moment van inwisseling. Bekijk het resultaat in de volgende video:
