# <a name="windows-10-for-the-enterprise-ways-to-use-devices-for-work"></a>Windows 10 for the enterprise: Ways to use devices for work (Engelstalig)
Windows 10 biedt u de mogelijkheid voor het benutten van Azure Active Directory (Azure AD). U kunt Windows 10-apparaten verbinden met Azure AD, zodat gebruikers bij Windows aanmelden zich met behulp van Azure AD-accounts of door de Azure-id's toe te voegen toegang te krijgen tot zakelijke apps en resources.
