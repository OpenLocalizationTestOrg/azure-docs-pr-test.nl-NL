Dit is een cumulatieve rapport waarin verdachte aanmeldingen vanaf de volgende rapporten:

<ul><li>Aanmeldingen vanaf onbekende bronnen</li>
<li>Aanmeldingen na meerdere mislukte pogingen</li>
<li>Aanmeldingen vanuit meerdere locaties</li>
<li>Aanmeldingen van IP-adressen met verdachte activiteit</li>
<li>Aanmeldingen vanaf mogelijk geïnfecteerde apparaten</li>
<li>Onregelmatige aanmeldingsactiviteiten</li>
</ul>
