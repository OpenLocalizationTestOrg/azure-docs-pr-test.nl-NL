# <a name="reset-a-password-for-a-user-in-the-azure-classic-portal"></a>Een wachtwoord voor een gebruiker in de klassieke Azure portal opnieuw instellen
> [!div class="op_single_selector"]
> * [Azure Portal](active-directory-users-reset-password-azure-portal.md)
> * [Klassieke Azure Portal](active-directory-create-users-reset-password.md)
>
>

Azure Active Directory (Azure AD) vereenvoudigt het proces van het wachtwoord van een gebruiker opnieuw instellen als ze vergeet en zien of tijdens het uitvoeren van beveiligingsonderhoud routinematige zijn vergrendeld. 

> [!IMPORTANT]
> Microsoft raadt u aan Azure AD te beheren met het [Azure AD-beheercentrum](https://aad.portal.azure.com) in Azure Portal in plaats van de klassieke Azure portal waarnaar in dit artikel wordt verwezen. Voor het opnieuw instellen van het wachtwoord van een gebruiker in de Azure AD-beheercentrum, Zie [opnieuw instellen van het wachtwoord voor een gebruiker in Azure Active Directory](active-directory-users-reset-password-azure-portal.md).

1. Open uw directory.
2. Selecteer de tab **Gebruikers** en selecteer vervolgens de weergavenaam van de gebruiker die u wilt wijzigen.
3. Selecteer in de opdrachtbalk **wachtwoord opnieuw instellen**.
4. Klik in het dialoogvenster van het wachtwoord opnieuw instellen op **opnieuw**.
5. Selecteer het selectievakje voor het voltooien van het wachtwoord opnieuw instellen.
