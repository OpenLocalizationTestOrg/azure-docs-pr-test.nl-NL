# <a name="delete-a-user-from-a-directory-in-azure-active-directory"></a>Een gebruiker vanuit een map in Azure Active Directory verwijderen
In dit artikel wordt uitgelegd hoe een gebruiker vanuit een map in Azure Active Directory (Azure AD) verwijdert. Zie voor meer informatie over het toevoegen van nieuwe gebruikers aan uw organisatie [nieuwe gebruikers toevoegen aan Azure Active Directory](active-directory-users-create-azure-portal.md).

## <a name="to-delete-a-user"></a>Een gebruiker verwijderen
1. Aanmelden bij [de Azure-portal](https://portal.azure.com) met een account met globale beheerdersrechten voor de map.
2. Selecteer **meer services**, voer **gebruikers en groepen** in het tekstvak in en selecteer vervolgens **Enter**.

   ![Gebruikersbeheer openen](./media/active-directory-users-delete-user-azure-portal/create-users-user-management.png)
3. Op de **gebruikers en groepen** blade Selecteer **gebruikers**.

   ![De blade gebruikers openen](./media/active-directory-users-delete-user-azure-portal/create-users-open-users-blade.png)
4. Selecteer op de blade **Gebruikers en groepen - Gebruikers** een gebruiker in de lijst.
5. Selecteer op de blade voor de geselecteerde gebruiker **overzicht**, en selecteer vervolgens in de opdrachtbalk **verwijderen**.

    ![Als u de opdracht Delete](./media/active-directory-users-delete-user-azure-portal/create-users-delete-command.png)
