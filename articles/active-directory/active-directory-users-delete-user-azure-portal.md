# <a name="delete-a-user-from-a-directory-in-azure-active-directory"></a>Een gebruiker vanuit een map in Azure Active Directory verwijderen
Dit artikel wordt uitgelegd hoe toodelete een gebruiker vanuit een map in Azure Active Directory (Azure AD). Zie voor meer informatie over het toevoegen van nieuwe gebruikers tooyour organisatie [toevoegen van nieuwe gebruikers tooAzure Active Directory](active-directory-users-create-azure-portal.md).

## <a name="toodelete-a-user"></a>een gebruiker toodelete
1. Aanmelden te[hello Azure-portal](https://portal.azure.com) met een account met globale beheerdersrechten voor Hallo-directory.
2. Selecteer **meer services**, voer **gebruikers en groepen** in het tekstvak Hallo en selecteer vervolgens **Enter**.

   ![Gebruikersbeheer openen](./media/active-directory-users-delete-user-azure-portal/create-users-user-management.png)
3. Op Hallo **gebruikers en groepen** blade Selecteer **gebruikers**.

   ![Blade openen Hallo-gebruikers](./media/active-directory-users-delete-user-azure-portal/create-users-open-users-blade.png)
4. Op Hallo **gebruikers en groepen - gebruikers** blade, selecteert u een gebruiker in de lijst Hallo.
5. Selecteer op de blade voor de geselecteerde gebruiker Hallo Hallo **overzicht**, en selecteer vervolgens in de opdrachtbalk Hallo **verwijderen**.

    ![De opdracht Delete Hallo selecteren](./media/active-directory-users-delete-user-azure-portal/create-users-delete-command.png)
