In dit onderwerp toont hoe tooenable Azure Active Directory: Identity Protection.

## <a name="steps-tooenable-azure-active-directory-identity-protection"></a>Stappen tooenable Azure Active Directory: Identity Protection
1. [Eenmalige aanmelding](https://ms.portal.azure.com/) tooyour Azure-portal als globale beheerder. 
2. Klik in hello Azure-portal, op **Marketplace**.
   
    ![Maak](./media/active-directory-identityprotection-enable/01.png "maken")
3. Klik in de lijst met de toepassingen van Hallo op **beveiliging en identiteit**.
   
    ![Maak](./media/active-directory-identityprotection-enable/02.png "maken")
4. Klik op **Azure AD Identity Protection**.
   
    ![Maak](./media/active-directory-identityprotection-enable/03.png "maken")
5. Op Hallo **Azure AD Identity Protection** blade, klikt u op **maken**.
   
    ![Maak](./media/active-directory-identityprotection-enable/04.png "maken")
