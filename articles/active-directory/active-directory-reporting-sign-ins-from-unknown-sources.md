# <a name="sign-ins-from-unknown-sources"></a>Aanmeldingen vanaf onbekende bronnen
Dit rapport geeft aan dat gebruikers die zich heeft aangemeld voor uw directory terwijl een client toegewezen IP-adres dat is herkend door Microsoft als een anonieme proxy IP-adres (bijvoorbeeld een Tor IP-adres). Deze proxy's worden vaak gebruikt door gebruikers die u wilt verbergen, IP-adres van de computer en kunnen worden gebruikt voor kwade bedoelingen.

Resultaten van dit rapport wordt het aantal keren dat een gebruiker aangemeld bij uw directory van dat adres en de proxy IP-adres weergegeven.
