# <a name="sign-ins-from-unknown-sources"></a>Aanmeldingen vanaf onbekende bronnen
Dit rapport geeft aan dat gebruikers die zich heeft aangemeld tooyour directory terwijl een client toegewezen IP-adres dat is herkend door Microsoft als een anonieme proxy IP-adres (bijvoorbeeld een Tor IP-adres). Deze proxy's worden vaak gebruikt door gebruikers die u wilt dat toohide IP-adres van de computer en kan worden gebruikt voor kwade bedoelingen.

De resultaten van dit rapport weergegeven Hallo aantal keren dat een gebruiker is aangemeld in de map tooyour van dat adres en het Hallo-proxy IP-adres.
