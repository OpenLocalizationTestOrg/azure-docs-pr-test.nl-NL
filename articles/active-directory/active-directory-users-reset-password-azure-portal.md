# <a name="reset-hello-password-for-a-user-in-azure-active-directory"></a>Hallo-wachtwoord opnieuw instellen voor een gebruiker in Azure Active Directory
> [!div class="op_single_selector"]

> * [Klassieke Azure Portal](active-directory-create-users-reset-password.md)
>
>

## <a name="how-tooreset-hello-password-for-a-user"></a>Hoe tooreset Hallo wachtwoord voor een gebruiker
1. Meld u aan toohello [Azure-portal](https://portal.azure.com) met een account met globale beheerdersrechten voor Hallo-directory.
2. Selecteer **meer services**, voer **gebruikers en groepen** in het tekstvak Hallo en selecteer vervolgens **Enter**.

   ![Gebruikersbeheer openen](./media/active-directory-users-reset-password-azure-portal/create-users-user-management.png)
3. Op Hallo **gebruikers en groepen** blade Selecteer **gebruikers**.

   ![Blade openen Hallo-gebruikers](./media/active-directory-users-reset-password-azure-portal/create-users-open-users-blade.png)
4. Op Hallo **gebruikers en groepen - gebruikers** blade, selecteert u een gebruiker in de lijst Hallo.
5. Selecteer op de blade voor de geselecteerde gebruiker Hallo Hallo **overzicht**, en selecteer vervolgens in de opdrachtbalk Hallo **wachtwoord opnieuw instellen**.

    ![Hallo selecteren opnieuw ingesteld wachtwoord](./media/active-directory-users-reset-password-azure-portal/create-users-reset-password-command.png)
6. Op Hallo **wachtwoord opnieuw instellen** blade Selecteer **wachtwoord opnieuw instellen**.
