# <a name="reset-the-password-for-a-user-in-azure-active-directory"></a>Het wachtwoord voor een gebruiker in Azure Active Directory opnieuw instellen
> [!div class="op_single_selector"]
> * [Azure Portal](active-directory-users-reset-password-azure-portal.md)
> * [Klassieke Azure Portal](active-directory-create-users-reset-password.md)
>
>

## <a name="how-to-reset-the-password-for-a-user"></a>Het wachtwoord voor een gebruiker opnieuw instellen
1. Aanmelden bij de [Azure-portal](https://portal.azure.com) met een account met globale beheerdersrechten voor de map.
2. Selecteer **meer services**, voer **gebruikers en groepen** in het tekstvak in en selecteer vervolgens **Enter**.

   ![Gebruikersbeheer openen](./media/active-directory-users-reset-password-azure-portal/create-users-user-management.png)
3. Op de **gebruikers en groepen** blade Selecteer **gebruikers**.

   ![De blade gebruikers openen](./media/active-directory-users-reset-password-azure-portal/create-users-open-users-blade.png)
4. Selecteer op de blade **Gebruikers en groepen - Gebruikers** een gebruiker in de lijst.
5. Selecteer op de blade voor de geselecteerde gebruiker de optie **Overzicht** en selecteer vervolgens in de opdrachtbalk **Wachtwoord opnieuw instellen**.

    ![Als u de opdracht Reset-wachtwoord](./media/active-directory-users-reset-password-azure-portal/create-users-reset-password-command.png)
6. Op de **wachtwoord opnieuw instellen** blade Selecteer **wachtwoord opnieuw instellen**.
