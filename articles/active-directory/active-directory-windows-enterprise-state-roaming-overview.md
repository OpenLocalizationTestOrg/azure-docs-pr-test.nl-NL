# <a name="enterprise-state-roaming-overview"></a><span data-ttu-id="abeaf-105">Overzicht van enterprise state roaming</span><span class="sxs-lookup"><span data-stu-id="abeaf-105">Enterprise State Roaming overview</span></span>
<span data-ttu-id="abeaf-106">Met Windows 10 [Azure Active Directory (Azure AD)](active-directory-whatis.md) gebruikers krijgen de mogelijkheid om te synchroniseren veilig hun gebruikersinstellingen en de instellingen van toepassingsgegevens naar de cloud.</span><span class="sxs-lookup"><span data-stu-id="abeaf-106">With Windows 10, [Azure Active Directory (Azure AD)](active-directory-whatis.md) users gain the ability to securely synchronize their user settings and application settings data to the cloud.</span></span> <span data-ttu-id="abeaf-107">Enterprise State Roaming biedt gebruikers een uniforme ervaring via hun Windows-apparaten en minder tijd nodig voor het configureren van een nieuw apparaat.</span><span class="sxs-lookup"><span data-stu-id="abeaf-107">Enterprise State Roaming provides users with a unified experience across their Windows devices and reduces the time needed for configuring a new device.</span></span> <span data-ttu-id="abeaf-108">Enterprise State Roaming werkt vergelijkbaar met de standaard [consumer instellingen sync](http://windows.microsoft.com/en-US/windows-8/sync-settings-pcs) die is geïntroduceerd in Windows 8.</span><span class="sxs-lookup"><span data-stu-id="abeaf-108">Enterprise State Roaming operates similar to the standard [consumer settings sync](http://windows.microsoft.com/en-US/windows-8/sync-settings-pcs) that was first introduced in Windows 8.</span></span> <span data-ttu-id="abeaf-109">Daarnaast biedt Enterprise State Roaming:</span><span class="sxs-lookup"><span data-stu-id="abeaf-109">Additionally, Enterprise State Roaming offers:</span></span>

* <span data-ttu-id="abeaf-110">**Scheiding van zakelijke en consumentgegevens** : organisaties zijn controle over hun gegevens en er is geen combinatie van zakelijke gegevens in een cloud-account van verbruiker of consumentgegevens in een enterprise cloud-account.</span><span class="sxs-lookup"><span data-stu-id="abeaf-110">**Separation of corporate and consumer data** – Organizations are in control of their data, and there is no mixing of corporate data in a consumer cloud account or consumer data in an enterprise cloud account.</span></span>
* <span data-ttu-id="abeaf-111">**Verbeterde beveiliging** : gegevens worden automatisch versleuteld vóór het verlaten van de gebruiker Windows 10-apparaten met behulp van Azure Rights Management (Azure RMS) en gegevens blijven versleuteld in rust in de cloud.</span><span class="sxs-lookup"><span data-stu-id="abeaf-111">**Enhanced security** – Data is automatically encrypted before leaving the user’s Windows 10 device by using Azure Rights Management (Azure RMS), and data stays encrypted at rest in the cloud.</span></span> <span data-ttu-id="abeaf-112">Alle inhoud blijft versleuteld in rust in de cloud, met uitzondering van de naamruimten, zoals namen van de instellingen en Windows-app.</span><span class="sxs-lookup"><span data-stu-id="abeaf-112">All content stays encrypted at rest in the cloud, except for the namespaces, like settings names and Windows app names.</span></span>  
* <span data-ttu-id="abeaf-113">**Beheer en controle voor een betere** – controle op en inzicht biedt ten opzichte van die instellingen in uw organisatie en op welke apparaten via de portal Azure AD-integratie wordt gesynchroniseerd.</span><span class="sxs-lookup"><span data-stu-id="abeaf-113">**Better management and monitoring** – Provides control and visibility over who syncs settings in your organization and on which devices through the Azure AD portal integration.</span></span> 

<span data-ttu-id="abeaf-114">Enterprise State Roaming is beschikbaar in meerdere Azure-regio's.</span><span class="sxs-lookup"><span data-stu-id="abeaf-114">Enterprise State Roaming is available in multiple Azure regions.</span></span> <span data-ttu-id="abeaf-115">U kunt de bijgewerkte lijst met beschikbare regio's vinden op de [Azure-Services per regio's](https://azure.microsoft.com/regions/#services) pagina onder Azure Active Directory.</span><span class="sxs-lookup"><span data-stu-id="abeaf-115">You can find the updated list of available regions on the [Azure Services by Regions](https://azure.microsoft.com/regions/#services) page under Azure Active Directory.</span></span>
