# <a name="azure-ad-domain-services---contact-us"></a>Azure AD Domain Services - contact met ons opnemen
## <a name="contact-hello-product-team"></a>Neem contact op met Hallo-productteam
Als u problemen met uw beheerde domein hebt, controleert u toosee als Hallo stappen die worden beschreven in Hallo [Troubleshooting Guide](active-directory-ds-troubleshooting.md) Hallo-probleem opgelost. Als u steeds problemen ondervindt nog, kunt u gratis toocontact ons.

U kunt dan een e-mail naar: [Azure AD Domain Services Feedback](mailto:aaddsfb@microsoft.com).

Zorg ervoor dat u de volgende hello, opneemt, zodat we Hallo probleem kunt onderzoeken.

* Hallo **tenant-ID en de map-ID** voor uw Azure AD-directory. Hallo tenant-ID is Hallo GUID die u in het veld voor de ID van Hallo directory op Hallo ziet [pagina Eigenschappen voor uw directory](https://ms.portal.azure.com/#blade/Microsoft_AAD_IAM/ActiveDirectoryMenuBlade/Properties)
* Hallo **DNS-domeinnaam** u hebt geconfigureerd voor uw beheerde domein met AAD Domain Services.

## <a name="provide-feedback"></a>Feedback geven
Uw feedback over Azure AD Domain Services via Hallo van harte Welkom  **[Azure Active Directory User Voice-kanaal](https://feedback.azure.com/forums/169401-azure-active-directory/)**.

Zorg ervoor dat u pre-behandeling uw vragen of feedback met Hallo woorden **'AADDS'**, voor het tooreach ons.
