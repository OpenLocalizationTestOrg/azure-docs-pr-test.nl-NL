# <a name="service-bus-messaging-samples"></a>Messaging service Bus-voorbeelden

Hallo messaging Service Bus-voorbeelden demonstreren belangrijke functies in [Service Bus-berichtenservice](https://azure.microsoft.com/services/service-bus/). U kunt op dit moment Hallo-voorbeelden zoeken op twee plaatsen:

- [Service Bus-berichtenservice voorbeelden op GitHub](https://github.com/Azure/azure-service-bus/tree/master/samples/DotNet): een nieuwe reeks voorbeelden, gehost op GitHub. Zie Hallo [Leesmij-bestand](https://github.com/Azure/azure-service-bus/blob/master/samples/DotNet/Microsoft.ServiceBus.Messaging/README.md) in Hallo-opslagplaats voor beschrijvingen van deze voorbeelden .NET. Hallo-voorbeelden worden voortdurend bijgewerkt, dus probeer back vaak voor updates.
- [Pagina met MSDN-voorbeelden](https://code.msdn.microsoft.com/site/search?query=service%20bus&f%5B0%5D.Value=service%20bus&f%5B0%5D.Type=SearchText&ac=5): oudere voorbeelden waarin Hallo MSDN woont code galerie. Hoewel deze voorbeelden is nog steeds werken, moet deze worden niet gehandhaafd en mogelijk verouderd met opzicht toocurrent aanbevolen procedures programmeren.
 
## <a name="service-bus-explorer"></a>Service Bus Explorer

Bovendien Hallo [Service Bus Explorer](https://github.com/paolosalvatori/ServiceBusExplorer) is een voorbeeld van een gehost op GitHub waarmee u tooconnect tooa Service Bus-Servicenaamruimte en eenvoudig beheren messaging-entiteiten. Hallo hulpprogramma biedt geavanceerde functies zoals import/export-functionaliteit en Hallo mogelijkheid tootest messaging-entiteiten en relay-services. U vindt Hallo volledige Service Bus Explorer bron- en -documentatie op [GitHub](https://github.com/paolosalvatori/ServiceBusExplorer).
