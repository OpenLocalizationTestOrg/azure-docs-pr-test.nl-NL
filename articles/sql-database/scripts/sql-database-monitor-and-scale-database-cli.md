## <a name="script-explanation"></a>Script uitleg

Dit script maakt gebruik van Hallo opdrachten te volgen. Elke opdracht in Hallo tabel koppelingen toocommand specifieke documentatie.

| Opdracht | Opmerkingen |
|---|---|
| [AZ groep maken](https://docs.microsoft.com/cli/azure/group#create) | Maakt een resourcegroep waarin alle resources worden opgeslagen. |
| [AZ sql server maken](https://docs.microsoft.com/cli/azure/sql/server#create) | Maakt een logische server die als host fungeert voor een database. |
| [AZ sql db weergeven-gebruik](https://docs.microsoft.com/cli/azure/sql/db#show-usage) | Bevat informatie over het gebruik Hallo grootte voor een database. |
| [AZ sql database-update](https://docs.microsoft.com/cli/azure/sql/db#update) | Updates van de database-eigenschappen (zoals Hallo laag of prestaties serviceniveau) of een database verplaatst naar buiten of tussen elastische pools. |
| [AZ groep verwijderen](https://docs.microsoft.com/cli/azure/vm/extension#set) | Hiermee verwijdert u een resourcegroep met inbegrip van alle ingesloten resources. |
