    <span data-ttu-id="ea2ae-156">Wanneer gegevens uiteindelijk aankomen, wordt de grafiek gaat terug naar een andere waarde dan nul.</span><span class="sxs-lookup"><span data-stu-id="ea2ae-156">When data eventually arrives, the chart jumps back to a non-zero value.</span></span> <span data-ttu-id="ea2ae-157">De waarschuwing wordt geëvalueerd op basis van de gegevens beschikbaar voor de periode die u hebt opgegeven.</span><span class="sxs-lookup"><span data-stu-id="ea2ae-157">The alert evaluates based on the data available for the period you specified.</span></span> <span data-ttu-id="ea2ae-158">Als het nieuwe gegevenspunt beschikbaar in de periode is, de statistische functie gebaseerd alleen op dat van gegevenspunt.</span><span class="sxs-lookup"><span data-stu-id="ea2ae-158">If the new data point is the only one available in the period, the aggregate is based just on that data point.</span></span>
