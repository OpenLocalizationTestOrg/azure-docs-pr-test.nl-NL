# <a name="supported-browsers-and-devices-for-the-azure-portal"></a>Ondersteunde browsers en apparaten voor de Azure-portal
U kunt uitvoeren de [Azure-portal](https://portal.azure.com) op alle moderne bureaublad Tablet PC-apparaten en browsers.

### <a name="supported-devices"></a>Ondersteunde apparaten
De Azure-portal fantastisch werkt op moderne pc's, Macs en tablets. Als u uw Azure-resources beheren vanaf een mobiel apparaat moet, probeert u de Azure-app die beschikbaar is voor het voorbeeld op iPhone- en Android. Zie voor meer informatie het blogbericht [introductie van de openbare preview van Azure app](https://azure.microsoft.com/blog/azure-app-preview/).

### <a name="supported-browsers"></a>Ondersteunde browsers
Het is raadzaam dat u de meest recente browser gebruiken die compatibel is met het besturingssysteem. De volgende browsers worden ondersteund:

* Microsoft Edge (meest recente versie)
* Internet Explorer 11
* Safari (meest recente versie, alleen Mac)
* Chrome (meest recente versie)
* Firefox (meest recente versie)
