# <a name="supported-browsers-and-devices-for-hello-azure-portal"></a>Ondersteunde browsers en apparaten voor hello Azure-portal
U kunt uitvoeren Hallo [Azure-portal](https://portal.azure.com) op alle moderne bureaublad Tablet PC-apparaten en browsers.

### <a name="supported-devices"></a>Ondersteunde apparaten
Hello Azure-portal fantastisch werkt op moderne pc's, Macs en tablets. Als u uw Azure-resources van een mobiel apparaat toomanage moet, probeer hello Azure-app die beschikbaar is voor het voorbeeld op iPhone- en Android. Zie voor meer informatie Hallo blogbericht [Introducing Hallo Apps van Azure openbare preview](https://azure.microsoft.com/blog/azure-app-preview/).

### <a name="supported-browsers"></a>Ondersteunde browsers
Het is raadzaam dat u Hallo meest recente browser gebruiken die compatibel is met het besturingssysteem. Hallo volgende browsers worden ondersteund:

* Microsoft Edge (meest recente versie)
* Internet Explorer 11
* Safari (meest recente versie, alleen Mac)
* Chrome (meest recente versie)
* Firefox (meest recente versie)
