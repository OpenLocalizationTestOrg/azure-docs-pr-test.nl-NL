# <a name="playing-your-content-with-existing-players"></a>Uw inhoud met bestaande spelers te spelen
Azure Media Services biedt ondersteuning voor veel populaire streaming-indelingen, zoals Smooth Streaming, HTTP Live Streaming en MPEG-Dash. In dit onderwerp verwijst tooexisting spelers waarmee u tootest uw stromen kunt.

### <a name="hello-azure-portal-media-services-content-player"></a>Hello Azure-portal Media Services content player
Hallo **Azure** portal biedt een speler inhoud die u tootest uw video gebruiken kunt.

Klik op Hallo gewenst video (Zorg ervoor dat deze werd [gepubliceerd](media-services-portal-publish.md)) en klik op Hallo **afspelen** knop onderaan Hallo Hallo-portal.

Hierbij geldt het volgende:

* Hallo **MEDIA SERVICES CONTENT PLAYER** speelt af vanaf Hallo standaard streaming-eindpunt. Als u tooplay vanaf een niet-standaard streaming-eindpunt wilt, gebruikt u een andere speler. Bijvoorbeeld: [Azure Media Player](http://amsplayer.azurewebsites.net/azuremediaplayer.html).

![AMSPlayer][AMSPlayer]

### <a name="azure-media-player"></a>Azure Media Player
Gebruik [Azure Media Player](http://amsplayer.azurewebsites.net/azuremediaplayer.html) tooplayback uw inhoud in een van de volgende indelingen hello (wissen of beveiligde):

* Smooth Streaming
* MPEG DASH
* HLS
* Progressieve MP4
