# <a name="community-portal"></a>Community-portal
## <a name="community-spotlight"></a>Community-spotlight
Laat het ons bevordering van uw project! Ons geweldig Hallo-project die u met Azure Cosmos DB hello volgende generatie Hallo DocumentDB stack werkt waarmee weergeven en wij helpen uw genius delen met Hallo wereld. toosubmit uw project Stuur ons een e-mailbericht op: [ askcosmosdb@microsoft.com ](mailto:askcosmosdb@microsoft.com).

### <a name="documentdb-lumenize"></a>documentdb-lumenize
*door Larry Maccherone*

Aggregaties (groeperen op, draaitabel en N dimensionale kubus) en Time Series transformaties als opgeslagen Procedures in Azure Cosmos DB DocumentDB API.

Afmelden controleren op [GitHub](https://github.com/lmaccherone/documentdb-lumenize) en [npm](https://www.npmjs.com/package/lumenize).

### <a name="documentdb-studio"></a>DocumentDB-Studio
*door Ming Liu*

Een client management viewer/explorer voor hello Azure Cosmos DB DocumentDB API-service.

Afmelden controleren op [GitHub](https://github.com/mingaliu/DocumentDBStudio).

### <a name="doqmentdb"></a>DoQmentDB
*door Ariel Mashraki*

DoQmentDB is een Node.js CTP-client, waarmee een MongoDB-achtige laag bovenop Azure Cosmos DB.

Afmelden controleren op [GitHub](https://github.com/a8m/doqmentdb) en [npm](https://www.npmjs.com/package/doqmentdb).

### <a name="typescript-api"></a>Machineschrift API
*door Jelmer Cormont*

Een wrapper rond Hallo Node.js-client is geschreven in machineschrift (werkt in gewone JavaScript te). Ondersteunt `async/await` en een vereenvoudigde API.

Afmelden controleren op [GitHub](https://github.com/jcormont/documentdb-typescript) en [npm](https://www.npmjs.com/package/documentdb-typescript).

### <a name="swagger-rest-api-for-documentdb"></a>Swagger REST-API voor DocumentDB
*door Howard Edidin*

Een bestand Azure Cosmos DB REST API Swagger voor Hallo DocumentDB-API die gemakkelijk kan worden geïmplementeerd als een API-App.

Afmelden controleren op [GitHub](https://github.com/HEDIDIN/DocumentDB-REST/tree/master/DocumentDBRestApi).

### <a name="fluent-plugin-documentdb"></a>beheersen op invoegtoepassing documentdb
*door Yoichi Kawasaki*

beheersen op invoegtoepassing documentdb is een invoegtoepassing Fluentd voor het uitvoeren van tooAzure Cosmos DB DocumentDB API.

Afmelden controleren op [GitHub](https://github.com/yokawasa/fluent-plugin-documentdb) en [rubygems](https://rubygems.org/gems/fluent-plugin-documentdb).

*Meer open-source Azure Cosmos DB projecten vinden op [GitHub](https://github.com/search?p=4&q=documentdb&type=Repositories).*
