## <a name="script-explanation"></a>Script uitleg

Dit script maakt gebruik van de volgende opdrachten. Elke opdracht in de tabel is gekoppeld aan de specifieke documentatie opdracht.

| Opdracht | Opmerkingen |
|---|---|
| [AZ groep maken](/cli/azure/group#create) | Maakt een resourcegroep waarin alle resources worden opgeslagen. |
| [AZ cosmosdb update](https://docs.microsoft.com/cli/azure/cosmosdb#az_cosmosdb_update) | Een account voor Azure DB die Cosmos-updates. |
| [AZ cosmosdb lijst-sleutels](https://docs.microsoft.com/cli/azure/cosmosdb#az_cosmosdb_list_keys) | Maakt een logische server die als host fungeert voor de SQL-Database. |
| [AZ groep verwijderen](https://docs.microsoft.com/cli/azure/group#az_group_delete) | Hiermee verwijdert u een resourcegroep met inbegrip van alle ingesloten resources. |
