# <a name="windows-universal-apps-sdk-release-notes"></a>Windows universele Apps SDK-releaseopmerkingen
## <a name="341-11032016"></a>3.4.1 (11/03/2016)

* Verbeteringen in stabiliteit.

## <a name="340-04192016"></a>3.4.0 (04/19/2016)
* Verbeteringen van de overlay bereiken.
* Logboeken van toegevoegde 'TestLogLevel' API inschakelen/uitschakelen/filter-console is verzonden door de SDK.
* Vaste activiteit in meldingen die gericht is op de eerste activiteit wordt niet weergegeven op App starten.

## <a name="331-02182016"></a>3.3.1 (02/18/2016)
* Vaste veroorzaakt een conflict tussen web-aankondiging HTML-inhoud en HTML-pagina van de SDK.
* Verbeteringen in stabiliteit.

## <a name="330-01222016"></a>3.3.0 (01/22/2016)
* Vaste crash opmaak van de UWP-apps in de releasemodus uitgevoerd.
* Vaste 1px marge op meldingen voor universele 8.1-apps.
* MS appx- en ms-appdata schema's beschikbaar op de actie-URL's.
* Verbeteringen in stabiliteit.

## <a name="320-11202015"></a>3.2.0 (11/20/2015)
* Ondersteuning toegevoegd voor Windows 10 Universal Windows Platform-toepassingen.
* Toegevoegde push kanaal functie om op te lossen kanaal conflicten (nu compatibel met Azure Notification Hubs) voor het delen.
* Vaste crashes bij het aanvragen van de apparaat-id direct na de initialisatie.
* Console-logboeken verbeteringen.
* Vaste crashes tijdens het parseren van een aantal onverwerkte uitzonderingen.

## <a name="310-05212015"></a>3.1.0 (05/21/2015)
* De Mobile Engagement apparaat-id is nu gebaseerd op een GUID die wordt gegenereerd tijdens de installatie.

## <a name="301-04292015"></a>3.0.1 (04/29/2015)
* Een fout die invloed hebben op de initialisatie van de SDK op sommige apps voor Windows Phone WinRT vast.

## <a name="300-04032015"></a>3.0.0 (04/03/2015)
* Introductie van de Mobile Engagement SDK voor universele App (Windows en Windows Phone WinRT).
* Het pictogram voor melding van de standaard wordt bijgewerkt.
* Back-systeem melding actie feedback verzenden wanneer een melding wordt geklikt.
* Vaste Systeemmelding die soms replay-app nadat wordt geklikt.
