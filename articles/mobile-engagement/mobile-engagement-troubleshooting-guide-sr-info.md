# <a name="service-request-sr-information"></a>Serviceaanvraaggegevens (SR)
Bieden de volgende informatie wanneer u een serviceaanvraag voor elke service Mobile Engagement opent Hallo:

## <a name="ids-applicable-identifiers-related-tooyour-issue"></a>Id's: Toepasselijke id's verwante tooyour probleem:
* App-ID
* Campagne-ID
* Apparaat-id
* Gebruikers-ID
* Gebruikersnaam
* Label van de App-gegevens

## <a name="errors-applicable-error-information"></a>Fouten: De betreffende foutgegevens
* Naam van Hallo-API of UI sectie waar Hallo probleem optreedt
* Tekst van een foutbericht verschijnt
* Resultaten van alle tests die u hebt uitgevoerd van Hallo [handleidingen voor het oplossen van problemen](http://go.microsoft.com/fwlink/?LinkId=524382)

## <a name="code-applicable-coding-information"></a>Code: De betreffende codering gegevens
* SDK-versie en het platform van uw app (Android SDK 2.4.1, iOS 1.16.2, enz.)
* Locatie van uw productie-app downloaden (of bestanden van uw app ontwikkelen APK/TGZ Hallo)
* 'AndroidManifest.xml' en/of een codefragment van uw app verwant tooAzure Mobile Engagement (voor geavanceerde probleemoplossing)
