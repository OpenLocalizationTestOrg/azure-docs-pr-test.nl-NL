# <a name="service-request-sr-information"></a>Serviceaanvraaggegevens (SR)
Geef de volgende informatie bij het openen van een serviceaanvraag voor elke service Mobile Engagement:

## <a name="ids-applicable-identifiers-related-to-your-issue"></a>Id's: Van toepassing-id's gerelateerd aan uw probleem:
* App-ID
* Campagne-ID
* Apparaat-id
* Gebruikers-ID
* Gebruikersnaam
* Label van de App-gegevens

## <a name="errors-applicable-error-information"></a>Fouten: De betreffende foutgegevens
* Naam van de API of UI sectie waar het probleem optreedt
* Tekst van een foutbericht verschijnt
* Resultaten van alle tests die u hebt uitgevoerd van de [handleidingen voor het oplossen van problemen](http://go.microsoft.com/fwlink/?LinkId=524382)

## <a name="code-applicable-coding-information"></a>Code: De betreffende codering gegevens
* SDK-versie en het platform van uw app (Android SDK 2.4.1, iOS 1.16.2, enz.)
* Locatie van uw productie-app (of de bestanden APK/TGZ van uw app ontwikkelen) downloaden
* 'AndroidManifest.xml' en/of een codefragment van uw app die betrekking hebben op Azure Mobile Engagement (voor geavanceerde probleemoplossing)
