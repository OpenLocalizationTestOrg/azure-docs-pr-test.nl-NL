<span data-ttu-id="3f0a7-113">Klik daarna in de rechterbovenhoek op **Ontwikkelaarsportal**.</span><span class="sxs-lookup"><span data-stu-id="3f0a7-113">Then click on **Developer portal** on the top-right.</span></span> 

![Koppeling naar de ontwikkelaarsportal in de uitgeversportal][api-management-pp-dp-link]

<span data-ttu-id="3f0a7-115">Als u de werkbalk voor aanpassing wilt openen, gaat u met de muis naar het aanpassingspictogram (of selecteer het pictogram) en klik op Stijlen op de werkbalk.</span><span class="sxs-lookup"><span data-stu-id="3f0a7-115">To open the customization toolbar move your mouse over the customization icon (or select it) and then click "styles" from the toolbar.</span></span>

<span data-ttu-id="3f0a7-117">Er zijn twee hoofdmanieren om stijlregels te bewerken. ![Werkbalk voor aanpassing][api-management-customization-toolbar]

<span data-ttu-id="3f0a7-119">Klik op de optie **Een element op de pagina selecteren** voor dit voorbeeld.</span><span class="sxs-lookup"><span data-stu-id="3f0a7-119">Click the **Select an element on the page** option for this example.</span></span>  <span data-ttu-id="3f0a7-120">Elementen worden nu gemarkeerd als u de muisaanwijzer er overheen beweegt om aan te geven welke stijlen van het element u zou gaan bewerken als u erop klikt.</span><span class="sxs-lookup"><span data-stu-id="3f0a7-120">Elements now become highlighted as you hover over them with the mouse to signify what element's styles you would start editing if you clicked.</span></span> <span data-ttu-id="3f0a7-121">Ga met de muisaanwijzer naar de tekst in de kop (normaal gesproken de bedrijfsnaam) en klik erop.</span><span class="sxs-lookup"><span data-stu-id="3f0a7-121">Move the mouse over the text in the header (typically you have the company name here), and then click it.</span></span> <span data-ttu-id="3f0a7-122">Een set benoemde en gecategoriseerde stijlregels wordt weergegeven in de stijleditor.</span><span class="sxs-lookup"><span data-stu-id="3f0a7-122">A set of named and categorized styling rules appears within the styling editor.</span></span> <span data-ttu-id="3f0a7-123">Elke regel geeft een stijleigenschap van het geselecteerde element aan.</span><span class="sxs-lookup"><span data-stu-id="3f0a7-123">Each rule represents a styling property of the selected element.</span></span> <span data-ttu-id="3f0a7-124">Voor de header die hierboven is geselecteerd, staat de tekengrootte bijvoorbeeld in @font-size-h1 terwijl de naam van het lettertype met alternatieven in @headings-font-family staat.</span><span class="sxs-lookup"><span data-stu-id="3f0a7-124">For example, for the header text selected above, the size of the text is in @font-size-h1 while the name of the font with alternatives is in @headings-font-family.</span></span>

> <span data-ttu-id="3f0a7-125">Als u bekend bent met [bootstrap][bootstrap]: deze regels zijn in feite [LESS-variabelen][LESS variables] binnen het bootstrapthema dat wordt gebruikt voor de ontwikkelaarsportal.</span><span class="sxs-lookup"><span data-stu-id="3f0a7-125">If you're familiar with [bootstrap][bootstrap], these rules are in fact [LESS variables][LESS variables] within the bootstrap theme used by the developer portal.</span></span>

<span data-ttu-id="3f0a7-126">Laten we de kleur van de koptekst wijzigen.</span><span class="sxs-lookup"><span data-stu-id="3f0a7-126">Let's change the color of the heading text.</span></span> <span data-ttu-id="3f0a7-127">Selecteer de invoer in het veld **@headings-color** en typ **#000000**.</span><span class="sxs-lookup"><span data-stu-id="3f0a7-127">Select the entry in the **@headings-color** field and type **#000000**.</span></span> <span data-ttu-id="3f0a7-128">Dit is de hexadecimale code voor de kleur zwart.</span><span class="sxs-lookup"><span data-stu-id="3f0a7-128">This is the hex code for the color black.</span></span> <span data-ttu-id="3f0a7-129">Terwijl u dit doet, ziet u dat er een vierkante kleurenindicator aan het einde van het tekstvak wordt weergegeven.</span><span class="sxs-lookup"><span data-stu-id="3f0a7-129">As you do this, you see that a square color indicator appears at the end of the text box.</span></span> <span data-ttu-id="3f0a7-130">Als u op deze indicator klikt, kunt u met een kleurenkiezer een kleur kiezen.</span><span class="sxs-lookup"><span data-stu-id="3f0a7-130">If you click this indicator, a color picker lets you to choose a color.</span></span>

![Kleurenkiezer][api-management-customization-toolbar-color-picker]

<span data-ttu-id="3f0a7-132">Wijzigingen worden weergegeven in realtime terwijl u ze toepast, maar zijn alleen zichtbaar voor beheerders.</span><span class="sxs-lookup"><span data-stu-id="3f0a7-132">Changes are previewed in real time as you make them, but are visible only to administrators.</span></span> <span data-ttu-id="3f0a7-133">Als u deze wijzigingen zichtbaar wilt maken voor iedereen, klikt u op de knop **Publiceren** in de stijleditor en bevestigt u de wijzigingen.</span><span class="sxs-lookup"><span data-stu-id="3f0a7-133">To make these changes visible to everyone, click the **Publish** button in the styling editor and confirm the changes.</span></span>

> <span data-ttu-id="3f0a7-135">Als u de stijlregels wilt wijzigen die van toepassing zijn op andere elementen op de pagina, volgt u dezelfde procedure als voor de header.</span><span class="sxs-lookup"><span data-stu-id="3f0a7-135">To change the style rules that apply to any other element on the page, follow the same procedure as you did for the header.</span></span> <span data-ttu-id="3f0a7-136">Klik op **Een element op de pagina kiezen** in de stijleditor, selecteer het betreffende element en begin met het wijzigen van de waarden van de stijlregels die op het scherm worden weergegeven.</span><span class="sxs-lookup"><span data-stu-id="3f0a7-136">Click **Select an element on the page** from the styling editor, select the element you are interested in, and start modifying the values of the style rules displayed on the screen.</span></span>
