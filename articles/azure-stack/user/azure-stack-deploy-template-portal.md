# <a name="deploy-templates-using-the-azure-stack-portal"></a>Sjablonen met behulp van de Stack van Azure-portal implementeren

*Van toepassing op: Azure Stack geïntegreerde systemen en Azure Stack Development Kit*

De portal gebruiken voor het implementeren van Azure Resource Manager-sjablonen op de Azure-Stack development kit.

Resource Manager-sjablonen implementeren en inrichten van alle resources voor uw toepassing in een enkele, gecoördineerde bewerking.

1. Aanmelden bij de portal, klikt u op **nieuw**, klikt u op **aangepaste**, en klik vervolgens op **sjabloonimplementatie**.
2. Klik op **template bewerken**, plak de code van uw JSON-sjabloon in de blade en klik vervolgens op **opslaan**.
3. Klik op **parameters bewerken**, typt u waarden voor de parameters die worden vermeld en klik vervolgens op **OK**.
4. Klik op **abonnement**, kies het abonnement dat u wilt gebruiken en klik vervolgens op **OK**.
5. Klik op **resourcegroep**, kiest u een bestaande resourcegroep of maak een nieuwe en klik vervolgens op **OK**.
6. Klik op **Create**. Een nieuwe tegel op het dashboard houdt de voortgang van de sjabloonimplementatie van uw.
