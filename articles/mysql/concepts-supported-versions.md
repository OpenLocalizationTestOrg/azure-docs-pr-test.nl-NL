# <a name="supported-azure-database-for-mysql-server-versions"></a>Ondersteunde Azure-Database voor MySQL server-versies
Azure MySQL-Database is ontwikkeld van MySQL Community Edition [MySQL Community Edition](https://www.mysql.com/products/community/), met Hallo InnoDB engine.  Azure MySQL-Database ondersteunt momenteel Hallo volgende versies.
