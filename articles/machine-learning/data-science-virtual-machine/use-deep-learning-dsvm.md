# <a name="using-the-deep-learning-virtual-machine"></a>Met behulp van de virtuele Machine Learning diep

Wanneer u uw virtuele Machine grondige Learning (DLVM) hebt ingericht, kunt u beginnen ontwikkelen van modellen diep neurale netwerk om samen te stellen AI-toepassingen in domeinen zoals computer visie, wat taal. 

Er zijn een aantal hulpprogramma's op de DLVM voor AI. De [grondige Learning en AI frameworks pagina](dsvm-deep-learning-ai-frameworks.md) bevat details over het gebruik van deze hulpprogramma's. De [pagina voorbeelden](dsvm-samples-and-walkthroughs.md) bevat verwijzingen naar de codevoorbeelden vooraf geladen op de virtuele machine voor elk van de frameworks kunt u snel aan de slag. 

# <a name="deep-learning-tutorials-and-walkthroughs"></a>Diep Learning zelfstudies en scenario's

Naast de voorbeelden op basis van het framework bieden we ook een set van uitgebreide scenario's die zijn gevalideerd op de DLVM. Deze help snel van de ontwikkeling van grondige learning-toepassingen in domeinen zoals installatiekopie en wat tekst/taal. We blijven meer end-to-end-zelfstudies toevoegen in verschillende domeinen en technologie.   

1. [Neural networks uitvoert via andere frameworks](https://github.com/ilkarman/DeepLearningFrameworks): dit is een uitgebreid overzicht waarin wordt uitgelegd hoe u de code van één framework naar de andere migreren. U ziet ook model vergelijken en snelheid uitvoeren via frameworks. 

2. [Een handleiding met instructies voor het bouwen van een end-to-end-oplossing om te detecteren producten binnen installatiekopieën](https://github.com/Azure/cortana-intelligence-product-detection-from-images): detectie van de installatiekopie is een techniek die kan vinden en classificeren van objecten binnen de installatiekopieën. Deze technologie loopt het risico op te zetten grote voordelen in veel praktijk zakelijke domeinen. Detailhandel kunnen bijvoorbeeld deze techniek gebruiken om te bepalen welke product een klant is opgehaald uit het plat. Deze informatie helpt op zijn beurt winkels productinventaris beheren. 

3. [Classificatie van documenten](https://github.com/anargyri/lstm_han): in dit scenario laat zien hoe u bouwen en training van twee verschillende neural netwerkarchitectuur (hiërarchische aandacht netwerk- en lang korte termijn geheugen (LSTM) netwerk) voor de classificatie van documenten met behulp van de API Keras voor grondige learning. Keras is een front-end tot drie van de meest populaire deep learning frameworks, Microsoft cognitieve Toolkit TensorFlow en Theano.
