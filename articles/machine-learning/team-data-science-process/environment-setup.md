# <a name="set-up-data-science-environments-for-use-in-the-team-data-science-process"></a>Gegevenswetenschapomgevingen instellen voor gebruik in Team Data Science Process
Het Team gegevens wetenschap proces maakt gebruik van verschillende gegevens wetenschappelijke omgevingen voor de opslag, verwerking en analyse van gegevens. Ze bevatten diverse typen virtuele machines in Azure, HDInsight (Hadoop)-clusters en werkruimten Azure Machine Learning in Azure Blob Storage. De beslissing over welke omgeving gebruiken afhankelijk van het type en de hoeveelheid gegevens die worden gemodelleerd en de bestemming voor deze gegevens in de cloud. 

* Zie voor instructies over de vragen in overweging moet nemen wanneer u deze beslissing, [van plan bent uw Azure Machine Learning gegevens wetenschappelijke omgeving](plan-your-environment.md). 
* Zie voor een catalogus van sommige scenario's die u tegenkomen kunt bij het uitvoeren van geavanceerde analyses [scenario's voor het Team gegevens wetenschap proces](plan-sample-scenarios.md)

Dit menukoppelingen naar onderwerpen waarin wordt beschreven hoe u de verschillende gegevens wetenschappelijke omgevingen die worden gebruikt door het Team gegevens wetenschap proces instelt.
