# <a name="copy-example-experiments-toocreate-new-machine-learning-experiments"></a>Kopieer bijvoorbeeld experimenten toocreate nieuwe machine learning-experimenten
Meer informatie over hoe toostart voorbeeld experimenten van [Cortana Intelligence Gallery](https://gallery.cortanaintelligence.com/) in plaats van machine learning-experimenten maken vanaf het begin. U kunt uw eigen machine learning-oplossing Hallo voorbeelden toobuild.

Hallo galerie heeft voorbeeld experimenten door Hallo Microsoft Azure Machine Learning-team als voorbeelden die worden gedeeld door Hallo Machine Learning-community. Hier kunt u ook vragen stellen of opmerkingen plaatsen over experimenten.

toosee hoe toouse Hallo gallery bekijkt hello 3 minuten durende video [kopiëren van anderen werk toodo voor gegevenswetenschap](machine-learning-data-science-for-beginners-copy-other-peoples-work-to-do-data-science.md) uit Hallo serie [Gegevenswetenschap voor beginnende gebruikers](machine-learning-data-science-for-beginners-the-5-questions-data-science-answers.md).

## <a name="find-an-experiment-toocopy-in-cortana-intelligence-gallery"></a>Een experiment toocopy niet vinden in de Cortana Intelligence Gallery
toosee welke experimenten zijn beschikbaar, gaat u toohello [galerie](https://gallery.cortanaintelligence.com/) en klik op **experimenten** bovenaan Hallo Hallo pagina.

### <a name="find-hello-newest-or-most-popular-experiments"></a>Hallo nieuwste of populairste experimenten zoeken
Op deze pagina kunt u zien **onlangs toegevoegd** experimenten of schuif omlaag toolook op **wat populair is** of de meest recente Hallo **populaire Microsoft-experimenten**.

### <a name="look-for-an-experiment-that-meets-specific-requirements"></a>Naar een experiment zoeken dat aan specifieke vereisten voldoet
toobrowse alle experimenten:

1. Klik op **door alle Bladeren** bovenaan Hallo Hallo pagina.
2. Aan de linkerkant Hallo en onder **verfijnen door** in Hallo **categorieën** sectie **Experiment** toosee alle experimenten in Hallo Hallo galerie.
3. U kunt op verschillende manieren experimenten vinden die aan uw vereisten voldoen:
   * **Selecteer filters op Hallo links.** Bijvoorbeeld: toobrowse experimenten die gebruikmaken van een PCA-gebaseerd anomaliedetectie-algoritme: met **Experiment** geselecteerde onder **categorieën**, klikt u op **Alles weergeven**. Kies vervolgens onder **Algorithms Used** de optie **PCA-Based Anomaly Detection**. <br></br>
     ![Filters selecteren](./media/machine-learning-sample-experiments/refine-the-view.png)
   * **Hallo zoekvak gebruiken.** Bijvoorbeeld: toofind experimenten die is bijgedragen door Microsoft met betrekking toodigit erkenning die gebruikmaken van een tweeklasse ondersteuning vector machine-algoritme 'cijferherkenning' in het zoekvak Hallo invoeren. Vervolgens selecteert filters Hallo **Experiment**, **Microsoft-inhoud**, en **Two-Class ondersteuning Vectormachine**:<br></br>
     ![Gebruik het zoekvak Hallo](./media/machine-learning-sample-experiments/search-for-experiments.png)
4. Klik op een experiment toolearn meer informatie.
5. toorun en/of Hallo experiment wijzigen, klikt u op **openen in Studio** op de pagina Hallo-experiment. <br></br>

    ![Voorbeeldexperiment](./media/machine-learning-sample-experiments/example-experiment.png)

    > [!NOTE]
    > Als u een experiment in Machine Learning Studio voor Hallo eerste keer opent, kunt u gratis proberen of een Azure-abonnement aanschaffen. [Meer informatie over de gratis proefversie van Machine Learning Studio versus betaalde service Hallo](https://azure.microsoft.com/pricing/details/machine-learning/)

## <a name="create-a-new-experiment-using-an-example-as-a-template"></a>Een nieuw experiment maken op basis van een sjabloon
U kunt ook een nieuw experiment in Machine Learning Studio maken op basis van een sjabloon uit de Galerie.

1. Meld u aan met uw Microsoft-account referenties toohello [Studio](https://studio.azureml.net), en klik vervolgens op **nieuw** toocreate een experiment.
2. Blader door Hallo voorbeeld inhoud en klikt u op een.

Een nieuw experiment gemaakt in uw Machine Learning Studio-werkruimte Hallo voorbeeld experiment als sjabloon gebruiken.
