# <a name="azure-machine-learning-studio-documentation"></a>Documentatie voor Azure Machine Learning Studio

Azure Machine Learning Studio is een hulpprogramma met functionaliteit op basis van slepen en neerzetten, waarmee u in samenwerkingsverband predictive analytics-oplossingen voor uw gegevens kunt ontwikkelen, testen en implementeren.
Zelfstudies, video's en voorbeeldmodellen laten zien hoe u Machine Learning-modellen kunt maken en implementeren met behulp van Studio.
