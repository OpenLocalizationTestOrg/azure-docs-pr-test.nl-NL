# <a name="managing-azure-web-app-using-azure-automation"></a>Azure Web-App met behulp van Azure Automation beheren
Deze handleiding vindt u de Azure Automation-service en hoe deze kan worden gebruikt voor het vereenvoudigen van beheer van Azure-Web-App.

## <a name="what-is-azure-automation"></a>Wat is Azure Automation?
[Azure Automation](../automation/automation-intro.md) is een Azure-service voor het cloudbeheer via procesautomatisering vereenvoudigen. Met behulp van Azure Automation, handmatige, kunnen herhaald, langlopende, foutgevoelige taken worden geautomatiseerd om betrouwbaarheid, efficiÃ«ntie en tijd voor de waarde voor uw organisatie te verhogen.

Azure Automation biedt een engine voor het uitvoeren van uiterst betrouwbare, maximaal beschikbare workflow die schaalbaar is om te voldoen aan uw behoeften. In Azure Automation kunnen processen worden gestarte handmatig, door systemen van derden 3rd of met regelmatige tussenpozen zodat taken gebeuren precies wanneer deze nodig is.

Operationele overhead verminderen en vrijmaken IT en DevOps-personeel te concentreren op het werk dat zakelijke meerwaarde door uw cloud-beheertaken automatisch wordt uitgevoerd door Azure Automation te verplaatsen.

## <a name="how-can-azure-automation-help-manage-azure-web-app"></a>Hoe kan Azure Automation helpen Azure-Web-App beheren?
Web-App in Azure Automation kan worden beheerd met behulp van de PowerShell-cmdlets die beschikbaar zijn in de [Azure PowerShell-modules](/powershell/azureps-cmdlets-docs). U kunt [deze Web-App PowerShell-cmdlets installeren in Azure Automation](https://azure.microsoft.com/blog/announcing-azure-resource-manager-support-azure-automation-runbooks/), zodat u al uw Web-App management-taken in de service kunt uitvoeren. U kunt ook deze cmdlets in Azure Automation met de cmdlets voor andere Azure-services tot complexe taken automatiseren via Azure-services en 3e systemen van derden koppelen.

Hier volgen enkele voorbeelden van het beheer van App-Services met Automation:

* [Scripts voor het beheren van Web-Apps](https://azure.microsoft.com/documentation/scripts/)
