# <a name="deploy-popular-application-frameworks-on-windows-using-azure-resource-manager-templates"></a>Implementatie van populaire toepassingsframeworks in Windows met behulp van Azure Resource Manager-sjablonen 

Werkbelastingen vereisen meestal veel bronnen werken volgens het ontwerp. Azure Resource Manager-sjablonen maken het mogelijk voor u niet alleen definiëren hoe toepassingen worden geconfigureerd, maar ook hoe de bronnen voor ondersteuning die zijn geconfigureerd voor toepassingen worden geïmplementeerd. Dit artikel vindt u de meest populaire sjablonen in de galerie en biedt u informatie over het gebruik van de Azure-portal, Azure CLI of PowerShell voor de implementatie ervan.
