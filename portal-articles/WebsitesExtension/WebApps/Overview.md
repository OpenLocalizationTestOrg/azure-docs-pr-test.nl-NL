# <a name="web-apps-documentation"></a>Documentatie voor Web Apps

Azure-Web-Apps kunt u toobuild en host webtoepassingen in Hallo programmeertaal van uw keuze zonder infrastructuurbeheer. Het biedt automatisch schalen en een hoge beschikbaarheid, ondersteuning voor zowel Windows als Linux en maakt automatische implementaties mogelijk vanuit GitHub, Visual Studio Team Services of een willekeurige Git-repo. Meer informatie over hoe toouse Azure Web Apps met onze snelstartgidsen, zelfstudies en voorbeelden.
