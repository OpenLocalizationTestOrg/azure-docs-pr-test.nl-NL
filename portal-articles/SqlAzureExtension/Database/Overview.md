# <a name="sql-database-documentation"></a>Documentatie voor SQL Database

Azure SQL-Database is een relationele database als een service met behulp van Hallo Microsoft SQL Server-Engine. SQL-Database is een krachtige, betrouwbare en veilige database kunt u toobuild gegevensgestuurde toepassingen en websites in Hallo programmeertaal van uw keuze, zonder toomanage-infrastructuur. Meer informatie over hoe toouse SQL Database met onze snelstartgidsen, zelfstudies en voorbeelden.
