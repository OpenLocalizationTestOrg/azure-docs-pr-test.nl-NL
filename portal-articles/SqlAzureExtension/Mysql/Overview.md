# <a name="azure-database-for-mysql-documentation"></a>Documentatie voor Azure Database voor MySQL

Azure MySQL-Database is een relationele database-service op basis van Hallo open-source-engine van MySQL-Server.  Deze volledig beheerde Database as a Service kan bedrijfskritieke workloads verwerken met voorspelbare prestaties en dynamische schaalbaarheid. Ontwikkel toepassingen met Azure-Database voor MySQL van Hallo open-source hulpprogramma's en platform van uw keuze. Meer informatie over hoe toouse Azure-Database voor MySQL met onze snelstartgidsen, zelfstudies en voorbeelden.
