# <a name="azure-database-for-postgresql-documentation"></a>Documentatie voor Azure Database for PostgreSQL

Azure Database for PostgreSQL is een relationele databaseservice op basis van de open-source PostgreSQL-database-engine. Deze volledig beheerde Database as a Service kan bedrijfskritieke workloads verwerken met voorspelbare prestaties, beveiliging, hoge beschikbaarheid en dynamische schaalbaarheid.  Ontwikkel toepassingen met Azure Database for PostgreSQL met behulp van open-sourcehulpprogramma's en een platform naar keuze.  Ontdek met onze snelstartgidsen, zelfstudies en voorbeelden hoe u Azure Database for PostgreSQL gebruikt.
