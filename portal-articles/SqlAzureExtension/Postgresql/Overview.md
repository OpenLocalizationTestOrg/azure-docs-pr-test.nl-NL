# <a name="azure-database-for-postgresql-documentation"></a>Documentatie voor Azure Database voor PostgreSQL

Azure PostgreSQL-Database is een relationele database-service op basis van Hallo open-source Postgres database-engine. Deze volledig beheerde Database as a Service kan bedrijfskritieke workloads verwerken met voorspelbare prestaties, beveiliging, hoge beschikbaarheid en dynamische schaalbaarheid.  Ontwikkel toepassingen met Azure-Database voor PostgreSQL van Hallo open-source hulpprogramma's en platform van uw keuze.  Meer informatie over hoe toouse Azure-Database voor PostgreSQL met onze snelstartgidsen, zelfstudies en voorbeelden.
