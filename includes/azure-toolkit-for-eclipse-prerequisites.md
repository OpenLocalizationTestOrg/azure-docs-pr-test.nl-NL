## <a name="prerequisites"></a>Vereisten
Hello Azure Toolkit voor Eclipse vereist Hallo softwareonderdelen te volgen:

* Eclipse IDE voor Java EE-ontwikkelaars, Mars, of later. U kunt dit downloaden van Hallo [Eclipse website](http://www.eclipse.org/downloads/).

* Een Java Developer Kit (JDK), versie 1.8 of hoger.

* Een besturingssysteem. Hello Azure Toolkit voor Eclipse is getest op Hallo volgende besturingssystemen:
  
  * Windows 10, Windows 8.1, Windows 8 en Windows 7
  * Windows Server 2012 R2, WindowsServer 2012, Windows Server 2008 R2 en WindowsServer 2008
  * [Mac OS X](http://www.apple.com/osx) versie 'Yosemite' en hoger
  * [Ubuntu Linux](http://www.ubuntu.com) versie 14, 15 en 16

Zie voor meer informatie, Hallo [Azure Toolkit voor Eclipse](http://marketplace.eclipse.org/content/azure-toolkit-eclipse) pagina op Hallo Eclipse Marketplace.
