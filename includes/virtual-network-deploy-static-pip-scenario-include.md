## <a name="scenario"></a>Scenario
Dit document begeleidt bij een implementatie die een statische openbare IP-adres is toegewezen aan een virtuele machine (VM) gebruikt. In dit scenario hebt u een enkele virtuele machine met een eigen statische openbare IP-adres. De virtuele machine deel uitmaakt van een subnet met de naam **FrontEnd** en heeft ook een statisch priv√© IP-adres (**192.168.1.101**) in dat subnet.
